# Titanic Survival Prediction and Data Analysis

This project performs data analysis, data cleaning, feature engineering, and machine learning on the Titanic dataset using Python.

## Features

- Load and explore Titanic dataset
- Check missing values and data types
- Analyze survival rate by:
  - Gender
  - Passenger class
  - Embarkation port
- Perform data cleaning
- Create new features:
  - FamilySize
  - IsAlone
  - Age_Group
- Convert categorical data into numeric format
- Build Logistic Regression model
- Predict passenger survival
- Evaluate model accuracy

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn

## Machine Learning

Model Used:
- Logistic Regression

## Dataset

Dataset file:
- Titanic_Project.xlsx

## Run Project

Install required libraries:

